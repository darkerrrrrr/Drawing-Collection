# お絵描きコレクション Bot

Discord 上でイラスト投稿を管理し、個人・全体のギャラリーを表示できる Bot です。

## 主な機能
- **自動保存**: 指定したチャンネルに投稿された画像を自動で保管庫へ転送します。
- **ギャラリー機能**: スライドショー形式で自分の作品や全体の作品を閲覧できます。
- **同期削除**: 元の投稿を削除すると、連動して保管庫からも削除されます。

## セットアップ方法

1.  **リポジトリのクローン**
2.  **依存ライブラリのインストール**
    ```bash
    pip install -r requirements.txt
    ```
3.  **環境変数の設定**
    `.env` ファイルを作成し、Bot のトークンを記述します。
    ```env
    DISCORD_TOKEN=your_token_here
    ```
4.  **Bot の起動**
    ```bash
    python main.py
    ```
5.  **初期設定**
    Discord 上で `/setup` コマンドを実行して、必要なチャンネルを作成してください。

## 技術スタック
- discord.py
- GitHub Actions (6時間サイクル実行)
- ステートレス設計 (Discord チャンネルを DB として活用)
